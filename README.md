# inmersia_learner_corpus
Linguistic analysis of an English learner corpus using spaCy and CEFR criterial features

## Project structure
- `notebooks/`: exploratory analysis and feature extraction notebooks
- `src/`: reusable Python scripts
- `outputs/`: figures and derived results
- `data/sample/`: sample data

## Main components
- preprocessing with spaCy
- lexical and syntactic feature extraction
- criterial features
- error analysis
- collocation analysis
